# Security

## Supported Use

This project is designed for:

- local report generation
- cloud-hosted model calls through your own provider key
- research memo creation from public-web evidence

## Reporting

If you find a security issue, please open a private report through GitHub Security Advisories if available, or contact the maintainer directly before public disclosure.

## Notes

- Do not commit API keys, reports, or `memory_palace/` snapshots.
- The repository intentionally ignores generated reports and local memory state.
- Public-web evidence can be noisy; do not treat output as financial advice or privileged diligence.
